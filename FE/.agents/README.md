# Frontend Subagents

이 폴더는 `TaskPulse Agent` 프런트엔드 팀의 로컬 서브에이전트 정의를 담는다.

구성 원칙:

- 각 서브에이전트는 화면/상태/품질 기준으로 역할을 분리한다.
- 사용자 행동 단위로 TDD 작업을 수행한다.
- API 연동은 mock 기반 테스트를 먼저 작성한다.

포함된 서브에이전트:

- `calendar-event-ux-engineer.md`
- `operations-ui-engineer.md`
- `quality-state-engineer.md`
